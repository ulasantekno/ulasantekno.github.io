## Langkah 2: Memahami Prompt Engineering

Prompt adalah **kunci sukses** di Midjourney. Format dasar:

```
/imagine prompt: [deskripsi gambar] --ar [aspect ratio] --v [version]
```

### Contoh Prompt yang Baik:
```
/imagine prompt: a futuristic cyberpunk city at night with neon lights, raining, cinematic lighting, hyperdetailed, 8k --ar 16:9 --v 6
```

### Element Penting dalam Prompt:
1. **Subject** - Apa yang ingin digambar
2. **Style** - Realistic, cartoon, anime, etc.
3. **Lighting** - Cinematic, soft, dramatic
4. **Details** - Hyperdetailed, 8k, intricate
5. **Parameters** --ar (aspect ratio), --v (version)

## Langkah 3: Generate Image Pertama

### Step-by-Step:
1. **Pilih channel** `#newbies` di Discord
2. **Ketik** `/imagine`
3. **Isi prompt** dengan deskripsi
4. **Tekan Enter** dan tunggu 60 detik
5. **Pilih gambar** dari 4 options yang dihasilkan

## Langkah 4: Upscale dan Variations

Setelah Midjourney generate 4 gambar:

### Options Available:
- **U1, U2, U3, U4** - Upscale gambar tertentu (higher resolution)
- **V1, V2, V3, V4** - Buat variations dari gambar tertentu
- **🔄** - Regenerate dengan prompt sama

### Tips Upscaling:
- **Upscale** untuk final artwork
- **Make Variations** untuk explore different styles
- **Regenerate** jika tidak puas dengan hasil

## Langkah 5: Advanced Techniques

### 1. Image Prompting
Upload gambar referensi + teks prompt:
```
/imagine prompt: [image_url] [deskripsi teks]
```

### 2. Style References
Gunakan `--sref` untuk konsisten style:
```
/imagine prompt: cat wearing sunglasses --sref [style_image_url]
```

### 3. Parameter Advanced:
- `--ar 2:3` - Portrait aspect ratio
- `--ar 16:9` - Widescreen aspect ratio  
- `--style raw` - Less opinionated style
- `--stylize 1000` - High creativity mode

## Contoh Projek Praktis

## Common Mistakes Pemula

### ❌ Jangan Lakukan:
1. **Prompt terlalu pendek** - "cat" vs "fluffy orange cat sleeping in sunlight"
2. **Lupa parameters** - Selalu specify --ar dan --v
3. **Gunakan channel wrong** - Gunakan #newbies untuk testing
4. **Ignore community guidelines** - Baca rules sebelum mulai

### ✅ Lakukan:
1. **Detail prompts** - Semakin detail, semakin baik
2. **Experiment** - Coba berbagai styles dan parameters
3. **Learn dari others** - Lihat prompt orang lain di Discord
4. **Save favorites** - Kumpulkan prompt yang bekerja baik
