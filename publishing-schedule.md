### Publishing Rules:
1. Always use WIB timezone (+0700)
2. Include timestamp untuk precise sorting
3. Rotate content types daily
4. Maintain consistent quality standards

### Notification Rules:
- Telegram notification untuk setiap publish

- Monthly report akhir bulan

- Jika quality score < 60: Pause publishing, review

- Jika algorithm changes: Pause, assess impact
