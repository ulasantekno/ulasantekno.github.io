## Automation Rules

### Content Generation:
- Use current date + timestamp untuk sorting
- Always include WIB timezone (+0700)
- Rotate content pillars daily
- Mix article lengths and formats

### Publishing Schedule:
- Morning articles: 08:00 WIB
- Afternoon articles: 14:00 WIB  
- Evening articles: 20:00 WIB
- Weekend: Flexible timing

### Notification Rules:
- Telegram notification untuk setiap publish
- Weekly performance summary setiap Senin
- Monthly report akhir bulan

## Emergency Protocols

### If Quality Drops:
- Reduce frequency temporarily
- Increase editing time
- Focus on pillar content only

### If Traffic Spikes:
- Monitor server performance
- Optimize images and caching
- Consider CDN jika perlu

### If Algorithm Changes:
- Pause publishing untuk assessment
- Analyze impact
- Adjust strategy accordingly
